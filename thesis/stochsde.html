Stochastic Processes and SDEs

A stochastic process is any process describing the evolution in time of a random phenomenon. biology, chemistry, ecology, neuroscience, physics, image processing, signal processing, control theory, information theory, computer science, and telecommunications.

Formal definition

A stochastic process is defined as a sequence of random variables on a common probability space \((\Omega, F, P)\) where \(\Omega \) is the sample space, F is a \(\sigma\)-algebra and P is a probability measure. The random variables indexed by some index set \(T\) (often the time) take values in a space \(S\) which is measurable respect to some \(\sigma\)-algebra

So for the given parameters a stochastic process can be written as:

\[ \{X(t):t\in T\} \]

Stochastic processes classification:

A stochastic process can be classified under two main categories:

  • Discrete time stochastic processes An example can be represented by random walks.
  • Continuous time stochastic processes An example can be represented by brownian motions.

Stochastic differential equations

A stochastic differential equation is a classical differential equation but it's perturbed by some random noise.

\[ dX_t = \mu(t, X_t) \, dt + \sigma(t, X_t) \, dW_t \]

Here, \(X_t\) is the stochastic process, \(\mu(t, X_t)\) is the drift term, \(\sigma(t, X_t)\) is the diffusion term, \(dt\) represents the differential of time, and \(dW_t\) represents the differential of a standard Wiener process (Brownian motion).

For given functions and a brownian motion \(W_t\), the solution to this equation is a function \(X\) that satisfies the following: \[ X(T) = \int_{0}^{T} \mu(t, X(t))dt + \int_{0}^{T} \sigma(t, X(t)) dW(t) \]

More in general, differential equation have common terms like:

  • Drift term: \(\mu(t, X_t)\) represents the deterministic part of the change in the process.
  • Diffusion term: \(\sigma(t, X_t)\) represents the random part of the change.
